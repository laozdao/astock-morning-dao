# 市场情绪指标构建方法：从行为金融到量化交易

> **一句话摘要**：基于行为金融学理论，构建涵盖市场广度、投资者行为、舆情数据的综合情绪指标体系，通过情绪极端值识别实现逆向投资和趋势确认。

---

## 一、引言

### 1.1 行为金融学视角

传统金融理论假设投资者是理性的，但行为金融学研究表明：

| 行为偏差 | 表现 | 市场影响 |
|----------|------|----------|
| 过度自信 | 高估自身判断能力 | 过度交易、泡沫形成 |
| 损失厌恶 | 对损失敏感度是收益的2倍 | 止损犹豫、处置效应 |
| 羊群效应 | 跟随大众决策 | 趋势强化、极端波动 |
| 锚定效应 | 过度依赖初始信息 | 价格粘性、反应不足 |
| 可得性偏差 | 过度关注近期事件 | 过度反应、波动放大 |

### 1.2 情绪周期理论

市场情绪遵循"贪婪-恐惧"周期：

```
绝望 → 怀疑 → 乐观 → 狂热 → 焦虑 → 恐慌 → 绝望
  ↑___________________________________________↓
```

**周期特征**：
- 每个阶段持续1-6个月不等
- 极端情绪往往预示反转
- 情绪拐点领先价格拐点1-2周

## 二、情绪指标体系

### 2.1 市场广度指标

#### 2.1.1 涨跌家数比

**定义**：上涨家数 / 下跌家数

| 区间 | 情绪状态 | 信号意义 |
|:----:|----------|----------|
| >5 | 极度贪婪 | 短期顶部风险 |
| 3-5 | 乐观 | 趋势延续 |
| 1-3 | 中性偏多 | 正常波动 |
| 0.5-1 | 中性偏空 | 谨慎观察 |
| 0.2-0.5 | 悲观 | 底部区域 |
| <0.2 | 极度恐惧 | 反弹机会 |

**历史统计（2019-2024）**：

| 情绪状态 | 出现次数 | 后续1周收益 | 后续1月收益 |
|----------|:--------:|:-----------:|:-----------:|
| 极度贪婪 | 23 | -1.2% | -2.5% |
| 极度恐惧 | 31 | +2.8% | +4.5% |

#### 2.1.2 涨跌停家数

| 指标 | 计算方法 | 极端值阈值 |
|------|----------|:----------:|
| 涨停家数 | 当日涨停股票数 | >150家（贪婪）|
| 跌停家数 | 当日跌停股票数 | >100家（恐惧）|
| 涨跌停比 | 涨停家数/跌停家数 | >10（贪婪），<1（恐惧）|

#### 2.1.3 创新高/新低家数

$$\text{情绪强度} = \frac{\text{创新高家数}}{\text{创新高家数} + \text{创新低家数}}$$

| 情绪强度 | 市场状态 | 操作建议 |
|:--------:|----------|----------|
| >80% | 强势市场 | 持有或加仓 |
| 50-80% | 健康市场 | 正常配置 |
| 20-50% | 弱势市场 | 降低仓位 |
| <20% | 极弱市场 | 等待反弹 |

### 2.2 投资者行为指标

#### 2.2.1 换手率

**定义**：成交额 / 流通市值

| 换手率水平 | 情绪状态 | 信号意义 |
|:----------:|----------|----------|
| >5% | 极度活跃 | 情绪高涨，警惕过热 |
| 3-5% | 活跃 | 趋势确认 |
| 1-3% | 正常 | 观望状态 |
| <1% | 低迷 | 情绪冰点，关注反转 |

#### 2.2.2 融资买入占比

$$\text{融资情绪} = \frac{\text{融资买入额}}{\text{市场成交额}} \times 100\%$$

| 融资情绪 | 杠杆情绪 | 信号意义 |
|:--------:|:--------:|----------|
| >12% | 极度乐观 | 杠杆资金激进 |
| 8-12% | 乐观 | 杠杆资金活跃 |
| 5-8% | 中性 | 正常水平 |
| <5% | 悲观 | 杠杆资金谨慎 |

#### 2.2.3 新开户数

新开户数反映散户入场意愿：

| 新开户变化 | 散户情绪 | 逆向信号 |
|:----------:|:--------:|----------|
| 环比+50% | 极度狂热 | 警惕顶部 |
| 环比+20% | 乐观 | 趋势确认 |
| 环比-20% | 悲观 | 底部区域 |
| 环比-50% | 极度悲观 | 反弹机会 |

### 2.3 衍生品市场指标

#### 2.3.1 期权PCR（Put/Call Ratio）

$$\text{PCR} = \frac{\text{Put成交量}}{\text{Call成交量}}$$

| PCR值 | 情绪状态 | 信号意义 |
|:----:|----------|----------|
| >1.2 | 极度恐惧 | 过度看跌，反弹机会 |
| 0.8-1.2 | 中性 | 正常波动 |
| <0.6 | 极度贪婪 | 过度看涨，回调风险 |

#### 2.3.2 波动率指数（VIX等效）

| VIX水平 | 市场情绪 | 操作建议 |
|:-------:|:--------:|----------|
| >40 | 极度恐慌 | 逆向买入 |
| 30-40 | 恐慌 | 逐步布局 |
| 20-30 | 中性 | 正常配置 |
| 15-20 | 平静 | 警惕波动 |
| <15 | 极度平静 | 警惕黑天鹅 |

### 2.4 舆情数据指标

#### 2.4.1 社交媒体情绪

**数据来源**：微博、雪球、东方财富股吧

| 指标 | 计算方法 | 情绪阈值 |
|------|----------|:--------:|
| 正面情绪占比 | 正面帖子/总帖子 | >70%（贪婪）|
| 负面情绪占比 | 负面帖子/总帖子 | >50%（恐惧）|
| 情绪极值 | 正面-负面 | >40（贪婪），<-40（恐惧）|

#### 2.4.2 搜索指数

**百度指数/微信指数**：

| 搜索变化 | 关注度 | 逆向信号 |
|:--------:|:------:|----------|
| 周环比+100% | 极度关注 | 短期顶部 |
| 周环比+50% | 高度关注 | 趋势后期 |
| 周环比-50% | 关注低迷 | 底部区域 |

## 三、综合情绪指数构建

### 3.1 指数构成

**Dao情绪指数（DEI）**构成：

| 成分指标 | 权重 | 数据来源 |
|----------|:----:|----------|
| 涨跌家数比 | 20% | 交易所 |
| 涨跌停比 | 15% | 交易所 |
| 换手率 | 15% | 交易所 |
| 融资买入占比 | 15% | 交易所 |
| PCR | 15% | 期权市场 |
| 舆情情绪 | 20% | 社交媒体 |

### 3.2 指数计算

**标准化处理**：

$$Z_i = \frac{X_i - \mu_i}{\sigma_i}$$

**加权求和**：

$$DEI = \sum_{i=1}^{6} w_i \times Z_i$$

**映射到0-100区间**：

$$DEI_{score} = 50 + 10 \times DEI$$

### 3.3 情绪区间划分

| DEI分数 | 情绪状态 | 市场含义 | 操作建议 |
|:-------:|:--------:|----------|----------|
| 80-100 | 极度贪婪 | 市场过热 | 减仓，逆向做空 |
| 60-80 | 贪婪 | 情绪高涨 | 分批止盈 |
| 40-60 | 中性 | 正常波动 | 持有观望 |
| 20-40 | 恐惧 | 情绪低落 | 逐步布局 |
| 0-20 | 极度恐惧 | 市场冰点 | 逆向买入 |

## 四、情绪驱动交易策略

### 4.1 逆向投资策略

**核心逻辑**：在别人恐惧时贪婪，在别人贪婪时恐惧

| 情绪状态 | 市场行为 | 逆向操作 | 历史胜率 |
|----------|----------|----------|:--------:|
| 极度贪婪（DEI>80） | 追涨买入 | 减仓/做空 | 65% |
| 极度恐惧（DEI<20） | 恐慌卖出 | 加仓/买入 | 72% |

**逆向投资条件**：

1. DEI达到极端值（<20或>80）
2. 持续3日以上极端状态
3. 成交量放大（确认情绪宣泄）
4. 无重大利空/利好干扰

### 4.2 趋势确认策略

**核心逻辑**：情绪与趋势共振时，趋势延续概率高

| 情绪状态 | 价格趋势 | 策略 | 胜率 |
|----------|----------|------|:----:|
| 贪婪+上涨 | 趋势延续 | 持有/加仓 | 70% |
| 恐惧+下跌 | 趋势延续 | 空仓/减仓 | 68% |
| 贪婪+下跌 | 趋势反转 | 观望 | 55% |
| 恐惧+上涨 | 趋势反转 | 观望 | 52% |

### 4.3 情绪动量策略

**情绪动量指标**：

$$\text{情绪动量} = DEI_t - DEI_{t-5}$$

| 情绪动量 | 信号 | 操作 |
|:--------:|------|------|
| >10 | 情绪快速升温 | 追涨（短期）|
| 5-10 | 情绪温和升温 | 持有 |
| -5至5 | 情绪平稳 | 观望 |
| -10至-5 | 情绪降温 | 减仓 |
| <-10 | 情绪快速降温 | 止损/抄底 |

## 五、情绪指标实战应用

### 5.1 大盘择时

**情绪择时信号**：

| 信号 | 条件 | 仓位建议 | 历史表现 |
|------|------|:--------:|----------|
| 买入信号 | DEI<20且开始回升 | 80-100% | 后续1月+5.2% |
| 减仓信号 | DEI>80且开始回落 | 20-40% | 规避-3.5%回调 |
| 持有信号 | DEI 40-60 | 60-80% | 正常波动 |

### 5.2 个股选择

**情绪过滤条件**：

1. 排除DEI>80时的热门股（避免追高）
2. 优选DEI<20时的优质股（逆向布局）
3. 关注情绪从极端值回归时的机会

### 5.3 行业轮动

**行业情绪差策略**：

$$\text{情绪差} = DEI_{行业} - DEI_{市场}$$

| 情绪差 | 策略 | 逻辑 |
|:------:|------|------|
| >20 | 减仓该行业 | 行业情绪过热 |
| <-20 | 加仓该行业 | 行业情绪冰点 |

## 六、风险控制

### 6.1 情绪陷阱

| 陷阱类型 | 特征 | 识别方法 | 应对措施 |
|----------|------|----------|----------|
| 情绪钝化 | 极端情绪持续但价格不动 | 观察价格反应 | 等待确认 |
| 假极端 | 单日极端后迅速回归 | 观察持续性 | 3日确认 |
| 结构性情绪 | 大盘中性但个股极端 | 分层分析 | 个股层面决策 |

### 6.2 情绪指标失效场景

| 场景 | 原因 | 应对 |
|------|------|------|
| 重大政策/事件 | 情绪被外部冲击主导 | 暂停情绪策略 |
| 流动性危机 | 情绪指标失真 | 关注资金指标 |
| 市场结构变化 | 投资者结构改变 | 重新校准参数 |

## 七、结论

### 7.1 核心要点

1. **情绪是可量化的**：通过多维度指标可以客观度量市场情绪
2. **极端情绪预示反转**：逆向投资在情绪极端时胜率最高
3. **情绪需要多维度验证**：单一情绪指标噪音大，需综合判断
4. **情绪策略有适用范围**：不适用于重大事件驱动的市场

### 7.2 应用建议

- **大盘择时**：DEI<20加仓，DEI>80减仓
- **个股选择**：避开情绪过热股，关注情绪冰点优质股
- **行业轮动**：利用行业情绪差进行配置调整
- **风险控制**：设置情绪钝化识别机制，避免假信号

### 7.3 未来研究方向

1. 高频情绪数据的应用（分钟级情绪指标）
2. 情绪指标与基本面数据的融合
3. 机器学习优化情绪权重
4. 跨市场情绪传导研究

### B. 情绪指标Python代码

```python
import pandas as pd
import numpy as np

def calculate_dei(data):
    """
    计算Dao情绪指数
    """
    # 标准化各成分
    z_adv = (data['涨跌家数比'] - data['涨跌家数比'].mean()) / data['涨跌家数比'].std()
    z_limit = (data['涨跌停比'] - data['涨跌停比'].mean()) / data['涨跌停比'].std()
    z_turn = (data['换手率'] - data['换手率'].mean()) / data['换手率'].std()
    z_margin = (data['融资买入占比'] - data['融资买入占比'].mean()) / data['融资买入占比'].std()
    z_pcr = (data['PCR'] - data['PCR'].mean()) / data['PCR'].std()
    z_sentiment = (data['舆情情绪'] - data['舆情情绪'].mean()) / data['舆情情绪'].std()
    
    # 加权求和
    weights = [0.20, 0.15, 0.15, 0.15, 0.15, 0.20]
    dei = np.dot([z_adv, z_limit, z_turn, z_margin, z_pcr, z_sentiment], weights)
    
    # 映射到0-100
    dei_score = 50 + 10 * dei
    return np.clip(dei_score, 0, 100)
```
