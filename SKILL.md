# A股全维度早盘综合分析报告 (Astock Morning DAO)

## 功能说明

本技能是六大维度早报系统的**总览技能**，负责：

1. **汇总**六大维度报告的核心结论
2. **交叉验证**各维度信号的一致性与矛盾
3. **综合研判**形成当日操作指导
4. **输出**可直接执行的交易策略

**⭐ 本技能为系统核心，输出当日最终操作结论**

### 信号表格行格式

每个维度的 `SIGNAL_TABLE_ROWS` 中一行格式：

```html
<tr>
  <td style="font-weight:600">维度名</td>
  <td><span class="tag tag-bull">利好</span></td>
  <td><span class="stars">⭐⭐⭐⭐</span></td>
  <td>核心结论描述</td>
</tr>
```

**Tag CSS类选择：**
- `tag-bull` — 利好/多头/流入/乐观
- `tag-bear` — 利空/空头/流出/悲观
- `tag-neutral` — 中性/分化
- `tag-inflow` — 资金流入
- `tag-outflow` — 资金流出

### 维度详情块格式

每个维度详情块格式：

```html
<div class="dim-detail">
  <h4>🌍 维度名（评分：XX/100 — 信号方向）</h4>
  <div class="data-grid">
    <div class="data-item">
      <div class="d-label">指标名</div>
      <div class="d-value up">数值</div>
      <div class="d-change up">变化描述</div>
    </div>
    <!-- 更多data-item... -->
  </div>
  <ul style="margin-top:12px">
    <li>分析要点一</li>
    <li>分析要点二，<strong>重点加粗</strong></li>
  </ul>
</div>
```

**data-value CSS类：** `up`（红涨）/ `down`（绿跌）/ 无类（中性）
**d-change CSS类：** `up` / `down` / 无类

### 生成流程

1. 读取 `assets/report-template.html`
2. 完成六大维度分析，收集数据
3. 将数据填入对应 `{{变量名}}` 占位符
4. 输出最终HTML文件到用户工作区

## 综合分析框架

### 一、六维度信号汇总表

每个维度输出一个标准化信号：

| 维度 | 信号方向 | 影响程度 | 核心结论（一句话） |
|------|----------|----------|-------------------|
| 宏观 | 多头/空头/中性 | ⭐1-5 | XXX |
| 政策 | 利好/利空/中性 | ⭐1-5 | XXX |
| 外围 | 利好/利空/中性 | ⭐1-5 | XXX |
| 资金 | 流入/流出/平衡 | ⭐1-5 | XXX |
| 情绪 | 乐观/悲观/中性 | ⭐1-5 | XXX |
| 技术 | 多头/空头/中性 | ⭐1-5 | XXX |

### 二、交叉验证矩阵

#### 高优先级交叉验证（权重更高）

| 组合 | 验证逻辑 | 共振信号 | 矛盾信号 |
|------|----------|----------|----------|
| **政策 × 资金** | 政策利好板块是否有资金流入确认 | 政策利好+北向加仓=高置信度 | 政策利好+资金流出=需警惕 |
| **宏观 × 政策** | 宏观周期与政策导向是否一致 | 复苏期+宽松政策=积极配置 | 衰退期+收紧政策=强烈避险 |
| **资金 × 情绪** | 资金流向与情绪指标是否匹配 | 北向流入+情绪乐观=确认趋势 | 北向流入+情绪悲观=分歧信号 |

#### 全维度共振判断

| 共振类型 | 条件 | 信号强度 | 操作建议 |
|----------|------|----------|----------|
| **六维共振看多** | ≥5个维度看多，其中3个高优先级看多 | ⭐⭐⭐⭐⭐ | 积极做多 |
| **四维共振看多** | 4个维度看多，含2个高优先级 | ⭐⭐⭐⭐ | 偏多操作 |
| **三维共振看多** | 3个维度看多，含1个高优先级 | ⭐⭐⭐ | 谨慎偏多 |
| **信号分歧** | 多空各半，无明确方向 | ⭐⭐ | 观望等待 |
| **三维共振看空** | 3个维度看空，含1个高优先级 | ⭐⭐⭐ | 谨慎偏空 |
| **四维共振看空** | 4个维度看空，含2个高优先级 | ⭐⭐⭐⭐ | 偏空操作 |
| **六维共振看空** | ≥5个维度看空，其中3个高优先级看空 | ⭐⭐⭐⭐⭐ | 强烈避险 |

### 三、矛盾处理规则

| 矛盾场景 | 处理原则 | 示例 |
|----------|----------|------|
| 高优先级 vs 低优先级 | 以高优先级为准 | 政策利好 vs 技术破位 → 跟随政策 |
| 同优先级矛盾 | 降低置信度，标记分歧 | 宏观利好 vs 资金流出 → 观望 |
| 多维度矛盾 | 暂缓操作，等待方向明确 | 四维看多两维看空 → 观望 |

## 综合评分体系

### 评分计算

```
综合评分 = Σ(维度评分 × 维度权重)

维度权重：
- macro-daily:      25%（高优先级）
- policy-daily:     25%（高优先级）
- flow-daily:       20%（高优先级）
- sentiment-daily:  12%（中优先级）
- global-daily:     10%（中优先级）
- technical-daily:   8%（中优先级）
```

### 评分→操作映射

| 综合评分 | 市场判断 | 仓位建议 | 操作策略 |
|----------|----------|----------|----------|
| **80-100** | 强烈看多 | 80-100% | 积极做多，重点配置共振板块 |
| **60-79** | 偏多 | 60-80% | 逢低布局，关注高优先级利好板块 |
| **40-59** | 中性 | 40-60% | 观望为主，轻仓操作 |

## 最佳实践

### 报告生成优先级

```
高优先级维度（必须详细分析）:
├── policy-daily  → 政策信号 + 受益板块
├── flow-daily    → 资金流向 + 热点识别
└── macro-daily   → 周期定位 + 大方向

中优先级维度（适度分析）:
├── sentiment-daily → 情绪极端值
├── global-daily    → 开盘指引
└── technical-daily → 关键点位
```

### 操作指导原则

1. **共振优先**: 多维度同向信号置信度最高
2. **高优先级优先**: 政策/资金/宏观权重更高
3. **矛盾降级**: 维度间矛盾时降低仓位
4. **止损纪律**: 每个操作必须设止损位
5. **不追高原则**: 回调介入优于追涨

### 仓位管理规则

| 综合评分 | 基础仓位 | 加仓条件 | 减仓条件 |
|----------|----------|----------|----------|
| 80-100 | 70% | 共振加强 | 评分降至60以下 |
| 60-79 | 50% | 评分升至80以上 | 评分降至40以下 |
| 40-59 | 30% | 方向明确 | 评分降至20以下 |
| 20-39 | 10% | 评分升至40以上 | 评分降至10以下 |
| 0-19 | 0% | 评分升至40以上 | - |

## 注意事项

1. **本报告为综合研判，不构成投资建议**
2. **高优先级维度结论权重更高**
3. **必须设置止损位，控制风险**
4. **开盘30分钟为关键观察期**
5. **尾盘确认持仓，评估次日策略**
6. **极端行情下（评分<20或>80）需格外谨慎**
7. **如遇节假日/非交易日，使用最近交易日数据**
